📚

محتوى منهجي متكامل

دروس مصممة خصيصاً للمنهج السعودي للصف الثاني ثانوي، تغطي جميع موضوعات الباب السادس — المتتابعات والمتسلسلات بشكل منظم ومتسلسل.

🤖

ذكاء اصطناعي متخصص

مدمج بتقنية Claude AI لتقديم شرح فوري وتصحيح مخصص لكل طالب، مع تحليل أسباب الأخطاء وتوجيهات تعليمية دقيقة.

📝

تدريب تفاعلي

أسئلة تدريبية متنوعة مع تصحيح ذكي وتغذية راجعة فورية تساعد الطالب على فهم أخطائه وتحسين أدائه قبل الاختبار.

⚡

مراجعة سريعة ومركزة

ملخص قوانين شامل وواضح في متناول يدك في أي وقت، مع فيديوهات مختارة لكل درس لمراجعة فعّالة وسريعة قبل الاختبارات.

ملخص القوانين — الباب السادس
المتتابعة الحسابية
الحد النوني
aₙ = a₁ + (n−1)d
الأساس d
d = أي حد − السابق له
الأوساط الحسابية
نحسب d من aₙ = a₁ + (n−1)d
المتتابعة الهندسية
الحد النوني
aₙ = a₁ · rⁿ⁻¹
الأساس r
r = أي حد ÷ السابق له
الأوساط الهندسية
نحسب r من aₙ = a₁ · rⁿ⁻¹
المتسلسلة الحسابية
المجموع (بالحد الأول والأخير)
Sₙ = n · (a₁ + aₙ) / 2
عدد الحدود في رمز Σ
n = آخر قيمة − أول قيمة + 1
المتسلسلة الهندسية
المجموع المنتهية
Sₙ = a₁(1−rⁿ) / (1−r)
الصيغة البديلة
Sₙ = (a₁ − aₙ·r) / (1−r)
غير المنتهية (|r|<1)
S = a₁ / (1−r)
المتسلسلة الهندسية غير المنتهية
متقاربة — |r| < 1
لها مجموع: S = a₁/(1−r)
متباعدة — |r| ≥ 1
ليس لها مجموع
